Setup Guide — Adobe Sign Admin Console

1

Log in to Adobe Sign as Admin

Go to Adobe Sign Admin Console and log in with the director's Adobe Sign account email.

2

Create Integration Key

Navigate to Account > Adobe Sign API > API Information, then click "Integration Key".

  • Name it: GodamWale ERP
  • Select ALL scopes (agreement_read, agreement_write, agreement_send, user_read, etc.)
  • Click Save
  • Copy the full key (it may extend beyond the visible area!)

Important: Integration Keys require an Enterprise or Developer subscription. If you don't see the option, contact Adobe support.

3

Paste Key Above & Save

Paste the Integration Key in the field above, fill in the director details, and click Save Settings. Then click Test Connection to verify.

4

Setup Webhook (Optional — for auto-status updates)

In Adobe Sign Admin Console: Account > Webhooks > Create

  • URL: {{ webhook_url }}
  • Events: Agreement All Signed, Agreement Recalled, Agreement Expired, E-Signed
  • Add the webhook secret to your server's .env file as ADOBE_SIGN_WEBHOOK_SECRET

Without a webhook, you can still manually sync agreement status from each agreement's detail page.

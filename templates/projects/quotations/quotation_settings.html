OAuth Setup Instructions

  1. Go to Google Cloud Console
  2. Create or select a project
  3. Enable Google Docs API and Google Drive API
  4. Go to "Credentials" → "Create Credentials" → "OAuth 2.0 Client ID"
  5. Select "Web application" as application type
  6. Add authorized redirect URI: {{ request.scheme }}://{{ request.get_host }}{% url 'projects:quotation_oauth_callback' %}
  7. Copy Client ID and Client Secret and paste above
  8. Save settings and authorize your Google account below
